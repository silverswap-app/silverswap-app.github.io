What is SilverSwap?

SilverSwap (also searched as 'silver swap') is a non-custodial decentralized exchange on Sonic for token swaps, concentrated liquidity, farms and AG ecosystem features. The official SilverSwap documentation covers tokenomics, emissions and the full feature set. Always verify the official domain before connecting a wallet.

How SilverSwap works

SilverSwap works through smart contracts on Sonic. Users connect a compatible wallet, choose a route and sign transactions from their own address.

  1. Select the Sonic networkUse a wallet configured for Sonic, then confirm that the app, chain and token contracts match the route you intend to trade.
  2. Review the quoteCheck input, output, price impact, minimum received and token names before approving or swapping.
  3. Sign wallet actionsApprovals and swaps are separate on-chain actions in many cases. Read each wallet prompt and avoid unlimited approvals when a smaller allowance fits.
  4. Track the resultCompleted transactions settle on Sonic. If a quote changes or fails, review the transaction status and route before trying again.

Fees and risk

There is no single SilverSwap fee that applies to every trade. Costs depend on Sonic network conditions, pool settings, token behavior and the exact route.

Cost components

Expect components such as Sonic gas, pool trading fees, price impact, approval transactions and token-specific transfer behavior. SilverSwap's tokenomics overview explains how swap fees fit into the protocol model.

Self-custody model

SilverSwap is used from a wallet, so you keep responsibility for keys, approvals and signatures. The official risk disclaimer tells users to understand DeFi and cryptocurrency risks before interacting.

Contract checks

Non-custodial trading still carries smart-contract, token, liquidity and phishing risk. Start with the official contract address page when verifying SilverSwap contracts.

Where SilverSwap fits on Sonic

SilverSwap is most relevant for traders already using Sonic assets or looking at AG, liquidity positions and Sonic-native DeFi routes.

Sonic base layer

Sonic is an EVM-compatible layer-1 network, and its native S token is used for network fees and staking. The Sonic documentation explains the S token's network role.

Concentrated liquidity

SilverSwap documents concentrated liquidity using Algebra technology, which lets liquidity providers choose active price ranges. Read the concentrated liquidity page before adding funds to a pool.

Audited contracts

SilverSwap lists Hashlock as its audit partner, but an audit is not a guarantee against bugs or losses. The Hashlock audit listing is the public source to review.

SilverSwap FAQ

Is silver swap the same as SilverSwap?

In crypto search, silver swap usually points to SilverSwap, the Sonic DEX at silverswap.io. The official styling is SilverSwap. Treat this page as an independent guide, and verify the domain before connecting a wallet.

What chain does SilverSwap use?

SilverSwap is built for Sonic. Sonic is an EVM-compatible network, and its S token is used for network activity such as fees and staking. The official S token docs explain the native token's role.

What can I do on SilverSwap?

You can use SilverSwap for token swaps, liquidity positions, selected farms, AG-related features and agS staking flows. Start with the official SilverSwap documentation, then check the live app for the current feature set and token availability.

What fees should I expect?

Expect components, not a fixed universal fee: Sonic gas, pool trading fees, price impact, approval transactions and token-specific behavior. SilverSwap's tokenomics page explains the protocol fee model at a high level, but the live quote matters before signing.

Is SilverSwap audited?

SilverSwap points users to Hashlock for audit information. You can review the public Hashlock SilverSwap listing, but an audit does not remove smart-contract, liquidity, token or user-signature risk.

What are the main risks?

The main risks are wallet phishing, wrong token contracts, unlimited approvals, volatile assets, thin liquidity, smart-contract bugs and misunderstanding AG or agS mechanics. SilverSwap's risk disclaimer says users should understand DeFi and cryptocurrency risks before interacting.
